Проверить команды
- ВЫБРАТЬ В ПРОВОДНИКЕ +++

- **ЛОКАЛЬНАЯ** (по номеру)+++

- С СЕРВЕРА (полная загрузка)

- С СЕРВЕРА ПО НОМЕРУ
	сразу загружаем файл в папку обмена
	загружен = истина
	если загружен - берем из папки обмена

- С СЕРВЕРА ПО НОМЕРУ + ФИЛИАЛ
	сразу загружаем файл в папку обмена

- ЗАГРУЗКА ИЗ ТАБЛИЧНОЙ ЧАСТИ
	Не включать загруженные файлы
	Удалить строку при загрузке
	Если не Загружен тогда загружаем

1. есть отбор (филиал)
2. Есть номер накладной+++
3. Нет ничего (полная загрузка)+++
	У меня будет полное имя файла в таблицу
	и Загружен = ложь
