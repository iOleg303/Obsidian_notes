# ТЕСТЫ 
- ВЫБРАТЬ В ПРОВОДНИКЕ +

- **ЛОКАЛЬНАЯ** (по номеру)

- С СЕРВЕРА (полная загрузка)

- С СЕРВЕРА ПО НОМЕРУ
	сразу загружаем файл в папку обмена
	загружен = истина
	если загружен - берем из папки обмена

- С СЕРВЕРА ПО НОМЕРУ + ФИЛИАЛ
	сразу загружаем файл в папку обмена

- ЗАГРУЗКА ИЗ ТАБЛИЧНОЙ ЧАСТИ
	Не включать загруженные файлы
	Удалить строку при загрузке
